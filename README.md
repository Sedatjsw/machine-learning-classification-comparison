# Machine Learning Classification Comparison using Bayesian Networks, Neural Networks, and Genetic Algorithms

This project explores multiple machine learning approaches for medical classification analysis using probabilistic modelling, neural networks, and feature selection techniques.

The project focuses on comparing different classification methodologies and analysing how feature dependency, optimisation, and feature selection influence predictive performance.

The workflow combines statistical analysis, Bayesian Networks, Neural Networks, and Genetic Algorithm-based feature selection within a complete machine learning pipeline.

---

## Project Overview

The main objective of this project is to investigate and compare different machine learning approaches for classification problems using a real-world medical dataset.

The project explores:

* Statistical dependency analysis
* Feature importance evaluation
* Bayesian Network classification
* Neural Network classification
* Genetic Algorithm feature selection
* Model comparison and evaluation

The analysis was performed using the Lymphography medical dataset.

## Dataset

The project uses the Lymphography Dataset containing medical classification data.

### Dataset Characteristics

* Multiple categorical medical features
* Multi-class classification problem
* Medical diagnosis-oriented dataset

The dataset was preprocessed using:

* Missing value handling
* Feature encoding
* Statistical dependency analysis
* Feature selection techniques

---

## Statistical Dependency Analysis

The project includes statistical analysis of relationships between features using:

* Chi-Square tests
* Cramér’s V analysis
* Mutual Information analysis

These techniques were used to identify the strongest feature relationships and evaluate variable dependency structures within the dataset.

---

## Bayesian Network Classification

Bayesian Networks were implemented to model probabilistic relationships between variables.

### Techniques Applied

* Bayesian structure learning
* PC Algorithm
* Conditional dependency analysis
* Probabilistic inference

The Bayesian Network approach allowed the project to analyse feature relationships beyond standard classification accuracy metrics.

---

## Neural Network Classification

A Neural Network classifier was also implemented and evaluated for comparison purposes.

### Neural Network Workflow

* Feature preprocessing
* Train/Test split
* Model training
* Classification prediction
* Performance evaluation

The Neural Network model was compared against probabilistic modelling approaches to evaluate predictive performance and generalisation.

---

## Genetic Algorithm Feature Selection

The project also applied Genetic Algorithms to optimise feature selection.

### Genetic Algorithm Objectives

* Reduce feature dimensionality
* Improve model efficiency
* Identify the most informative features
* Optimise classification performance

Feature subsets were evaluated iteratively to identify combinations that improved predictive performance.

---

## Model Evaluation

Multiple evaluation techniques were used throughout the project, including:

* Classification accuracy
* Feature importance analysis
* Dependency analysis
* Comparative model evaluation

The workflow focused not only on predictive performance, but also on understanding feature interactions and model behaviour.

## Key Insights

* Statistical dependency analysis revealed strong relationships between several medical variables
* Bayesian Networks provided interpretable probabilistic relationships between features
